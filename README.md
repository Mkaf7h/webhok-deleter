## 🛠️ HOW TO USE | طريقة الاستخدام

1.  **Paste** the Target Webhook URL into the input field.
2.  **Delete:** Click the `DELETE WEBHOOK` button to wipe it from Discord servers.
3.  **Spam:** Enter your message and hit `START SPAM` to flood the channel.
